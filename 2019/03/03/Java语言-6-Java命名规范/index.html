6-Java命名规范

发表于 2019-03-03 | 分类于 做一个有骨气的Coder ， 我的JavaEE哪有这么熟练 ， Java语言 ， 入门篇

包（Package）的命名

Package的名字应该都是由一个小写单词组成，一般可以命名为“com.公司名.项目名.模块名….”，例如：com.sun.swt.……等。

变量的命名

变量的名字可以大小写混用，但首字符应小写。词由大写字母分隔，限制使用下划线（_）或美元符号（$），因为这个字符对内部类有特殊含义，例如：idCard等。

常量（final变量）的命名

final变量的名字应该都大写，单词与单词以_分隔，并且指出完整的含义，例如：MAX_VALUE等。

类（Class）的命名

Class的名字通常由多个单词合成，要求每个单词的首字母都要大写，例如：HelloWorld等。

接口（Interface）的命名

与Class的命名类似。

数组的命名

数组应该总是用下面的方式来命名：byte[] buffer;，而不是byte buffer[];。

方法的命名

方法名的第一个单词应该是动词，大小写可以混用，但首字符应小写。在每个方法名内，大写字母将词分隔并限制使用下划线。参数的名字必须和变量的命名规范一致。使用有意义的参数命名，如果可能的话，使用和要赋值的字段一样的名字：
