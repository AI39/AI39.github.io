数据类型

基本类型 — 用于保存简单的单个数据

整型

整型用于表示没有小数部分的数值，它允许是负数。

类型 存储需求 取值范围
int 4字节 -2 147 483 648 ~ 2 147 483 647（正好超过20亿）
short 2字节 -32 768 ~ 32 767
long 8字节 -9 223 372 036 854 775 808 ~ 9 223 372 036 854 775 807
byte 1字节 -128 ~ 127

注意：

  • Java中所有的数值类型所占据的字节数量与平台无关。
  • Java中没有任何无符号（unsigned）形式的int、long、short或byte类型。

浮点型

浮点型用于表示有小数部分的数值。

类型 存储需求 取值范围
float 4字节 大约±3.402 823 47E+38F(有效数字为6~7位)
double 8字节 大约±1.797 693 134 862 315 70E+308(有效数字为15位)

下面是用于表示溢出和出错情况的三个特殊的浮点数值：

  • Double.POSITIVE_INFINITY：正无穷大
  • Double.NEGATIVE_INFINITY：负无穷大
  • Double.NaN：不是一个数，如计算0/0或负数的平方根。

注意：

  • double表示这种类型的数值精度是float类型的两倍，绝大部分应用程序都采用double类型。在很多情况下，float类型的精度很难满足需求。且float类型的数值有一个后缀F或f，没有后缀的浮点数值默认为double类型。

字符型

char类型原本用于表示单个字符。现在，有些Unicode字符可以用一个char值描述，另外一些Unicode字符则需要用两个char值描述。

类型 存储需求
char 2字节

注意：

  • char类型的字面量值要用单引号括起来。例如：’A’的编码值为65所对应的字符常量。它与”A”不同，”A”是包含一个字符A的字符串。char类型的值可以表示为十六进制值，其范围从\u0000到\Uffff。
  • 在Java中，char类型描述了UTF-16编码中的一个代码单元。我们强烈建议不要在程序中使用char类型，除非确实需要处理UTF-16代码单元，最好将字符串作为抽象数据类型处理（有关内容见下面的String部分）。

变量

变量的定义：

变量就是计算机中一块特定的内存空间，由一个或多个连续的字节组成。

在Java中，每个变量都有一个类型，变量的类型决定了为它分配多少个字节空间以及它能变化的范围。

变量的声明和初始化：

变量的声明：数据类型 变量名; 例如：int value;

变量的初始化：变量名 = 值; 例如：value = 0;

注意：

  • 声明一个变量以后，必须用赋值语句对变量进行显示初始化，使用未初始化的变量Java编译器会报错。例如：

变量的命名规范

  • 变量名只能以字母、数字、_符号和$符号组成。
  • 第一个字符只能是字母_符号和$符号，不能是数字。
  • 变量名不能包含除_符号和$符号以外的任何特殊字符，如%、#等。
  • 不可以使用关键字作为变量名，如不能把变量名取作int。
